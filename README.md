# 📦 Inventory Manager (Python CLI)

A simple command-line inventory manager built with Python.  
This project allows users to manage products and quantities through a terminal interface.

The goal of this project was to practice:
- Python functions
- Dictionaries
- File handling
- Modular programming
- Basic error handling

---

# 🚀 Features

The program allows you to:

- Add a product with its quantity
- Update the quantity of an existing product
- Show the full inventory
- Delete a product
- Search for a product
- View inventory statistics
- Save and load inventory from a file

---

# 🧠 Inventory Statistics

The program can calculate:

- Total number of different products
- Total number of items in stock
- Product with the highest stock quantity
- Product with the lowest stock quantity

# 📂 Project Structure

```
python-inventory-manager/
│
├── main.py
├── inventory_manager.py
├── inventory.txt
└── README.md
```

- **main.py** → Handles the user interface and menu
- **inventory_manager.py** → Contains all inventory functions
- **inventory.txt** → Stores the saved inventory data

# ▶️ How to Run

1. Clone the repository

git clone https://github.com/ElfenCamii/python-inventory-manager.git

2. Go to the project folder

cd python-inventory-manager

3. Run the program

python main.py
